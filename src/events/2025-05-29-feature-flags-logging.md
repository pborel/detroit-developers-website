A two-presentation meetup in partnership with DevOps Detroit. The first talk covered building a feature flagging system using Node, React, and TypeScript, led by RIVET's CTO Andy Lawrence. The second by cloud engineer Umelo Onyejiaka detailed deploying logging infrastructure to AWS EKS with Kubernetes, bastion host implementation, and Helm configuration management.
