Design Notes

  • Color palette: Catppuccin Mocha — #1e1e2e bg, #181825 cards, #313244 borders, #cdd6f4 text
  • Accent colors: green (#a6e3a1) comments, blue (#89b4fa) links/keys, pink (#f38ba8) highlights, peach (#fab387) file icons
  • Navigation: Sidebar file explorer (pages as files, blog posts as files in posts/ folder), editor tabs along top, breadcrumb path
  • Line numbers: Decorative gutter along the left side of content
  • Status bar: VS Code-style bar at bottom with branch name and file info
  • Code comments: // NEXT EVENT, // README, // RECENT POSTS as section labels
  • Responsive concern: Sidebar collapses on mobile — could become a top nav or slide-out drawer
