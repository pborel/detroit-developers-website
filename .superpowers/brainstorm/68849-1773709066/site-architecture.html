Site Architecture

11ty static site with CLI/IDE aesthetic

Project Structure

detroit-developers-website/
  ├── src/
  │  ├── _includes/      # Nunjucks layouts & partials
  │  │  ├── base.njk     # HTML shell, nav, footer
  │  │  ├── post.njk     # Blog post layout
  │  │  └── event.njk    # Past event layout
  │  ├── _data/         # 11ty global data files
  │  │  └── meetup.js    # Build-time scraper for upcoming events
  │  ├── posts/         # Blog posts (Markdown)
  │  ├── events/        # Past events (Markdown + frontmatter)
  │  ├── assets/        # CSS, images, fonts
  │  ├── index.njk     # Home page
  │  ├── events.njk    # Events listing page
  │  ├── blog.njk      # Blog listing page
  │  └── about.njk     # About page
  ├── .eleventy.js        # 11ty config
  └── package.json

Data Flow

# Upcoming events (automatic)
Build → meetup.js scrapes meetup.com/detroit-developers/
      → extracts event JSON-LD / HTML
      → available as {{ meetup.upcoming }} in templates
      → fallback: "Visit us on Meetup" link
# Past events (manual Markdown)
Author → writes events/2026-03-agentic-panel.md
        → frontmatter: title, date, video, photos
        → auto-listed on /events page
# Blog posts (manual Markdown)
Author → writes posts/my-post.md
        → auto-listed on /blog page

Pages

/ (Home)
  • Hero: next upcoming event
  • About blurb
  • Recent blog posts (3)
  • CTA: Join on Meetup
/events
  • Upcoming event(s) from scraper
  • Past events archive (cards with video/photo thumbnails)
/blog
  • Post listing (title, date, excerpt)
  • Individual post pages
/about
  • Community description
  • Organizer info
  • Links: Meetup, LinkedIn
