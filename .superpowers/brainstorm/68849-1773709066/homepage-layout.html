Design Notes

  • Color palette: GitHub dark — #0d1117 bg, #161b22 cards, #21262d borders, #f0f6fc text
  • Accent colors: green (#7ee787) for prompts, blue (#58a6ff) for links, light blue (#79c0ff) for metadata, red (#ff7b72) for highlights
  • Typography: Monospace (Fira Code / JetBrains Mono) for CLI elements, system sans-serif for body content
  • CLI metaphor: Each section opens with a terminal command ($ next-event, $ cat README.md, $ ls posts/) as a stylistic flourish — not functional
  • Responsive: Single column on mobile, nav collapses to hamburger
